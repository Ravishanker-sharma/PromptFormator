# Prompt Formatter Extension

Write beautiful, structured prompts for AI chatbots (ChatGPT, Claude, Gemini, etc.) directly in your browser. This extension provides a seamless Rich Text Editor that automatically converts your formatting into perfect Markdown.

## Features

-   **Rich Text Styling**: Visualize **Bold**, *Italics*, and Lists as you type.
-   **Headings**: Easily add structure with H1 (`#`), H2 (`##`), and H3 (`###`).
-   **Indent with Tab**: Pressing `Tab` inserts 4 spaces (instead of changing focus).
-   **Markdown Conversion**: Your rich text is automatically converted to Markdown when inserted.
-   **Privacy Focused**: The editor isolates your keystrokes so they don't leak to the underlying chat box until you hit "Insert".

## Installation

1.  **Download** this repository to your computer.
2.  Open Chrome (or Brave/Edge) and navigate to `chrome://extensions`.
3.  Toggle **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the folder containing these files.

## Usage

1.  Navigate to any AI chat interface (e.g., chatgpt.com).
2.  **Focus** the main text input area.
3.  A **Floating Pencil Button** will appear near the cursor/input (or use `Alt + A`).
4.  Click it to open the **Prompt Editor**.
5.  Write your prompt using the toolbar or shortcuts:
    -   **Bold**: `Cmd/Ctrl + B`
    -   **Italic**: `Cmd/Ctrl + I`
    -   **List**: `Cmd/Ctrl + Shift + L` (Mac)
6.  Click **Insert Prompt** to send the formatted text to the chat box.

## Troubleshooting

-   **Button not appearing?** Try clicking inside the text box again or refreshing the page.
-   **Shortcuts not working?** Ensure the editor overlay is focused.
